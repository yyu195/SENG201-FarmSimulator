

Run program in Eclipse:  Run the program from the FarmManager class
1.File->New->Java Project(right click) Import->General->File System then choose FarmSimulator
2.Create an eclipse project in the directory:create a java project
Uncheck the Use default location option, and choose the directory where is your non-eclipse project
Click Next and Configure the sub-directories of your non-Eclipse project where
The source files are located.


Run Program in Terminal:
1.unzip yyu71_jli310_zip
2. Run “java -jar yyu71_jli310_FarmSimulator.jar”
